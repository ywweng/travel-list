# Travel List

## 專案介紹
使用原生 JavaScript + MVC 架構打造的旅行清單，可以規劃一年內的行程，最長天數為 18 天。使用者可以添加卡片，並在卡片中添加行程，可任意拖曳、編輯、刪除每個行程。

## 功能
- 使用者選擇日期並產生卡片
- 使用者可以刪除卡片
- 使用者可以在卡片中添加行程
- 使用者雙擊行程項目以編輯行程
- 使用者可以拖曳行程至任何卡片
- 使用者可以刪除行程
- 所有行程資料儲存於 localStorage

## 安裝專案
1. Clone 專案至本地
   ```
   git clone https://github.com/ywweng/travel-list.git
   ```
2. 開啟 index.html
